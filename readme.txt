YouTube Shorts videolarını şık bir swiper slider içinde görüntüleyin.

== Description ==

YouTube Shorts Swiper, YouTube Shorts videolarınızı web sitenizde modern ve responsive bir slider içinde görüntülemenizi sağlar.

**Özellikler:**

* Kolay video yönetimi - Admin panelden video ekle/sil/düzenle
* Otomatik thumbnail çekme - YouTube'dan otomatik
* Responsive tasarım - Desktop, tablet ve mobil için ayrı ayarlar
* Tıkla-oynat - Sayfada direkt video oynatma
* Autoplay slider - Otomatik kaydırma özelliği
* Shortcode desteği - Herhangi bir sayfaya ekleyin

**Kullanım:**

1. WordPress admin panelinde "YT Shorts Swiper" menüsüne gidin
2. Video URL'lerini ekleyin
3. Ayarları özelleştirin
4. `[yt_shorts_swiper]` shortcode'unu istediğiniz sayfaya ekleyin

== Installation ==

1. Eklenti dosyalarını `/wp-content/plugins/yt-shorts-swiper` klasörüne yükleyin
2. WordPress admin panelinden eklentiyi aktif edin
3. "YT Shorts Swiper" menüsünden ayarları yapılandırın

== Frequently Asked Questions ==

= Hangi YouTube URL formatları destekleniyor? =

* https://youtube.com/shorts/VIDEO_ID
* https://youtu.be/VIDEO_ID
* https://youtube.com/watch?v=VIDEO_ID

= Kaç video ekleyebilirim? =

Sınır yoktur, istediğiniz kadar video ekleyebilirsiniz.

= Thumbnail'lar nereden geliyor? =

Thumbnail'lar YouTube'dan otomatik olarak çekilir, API key gerekmez.
