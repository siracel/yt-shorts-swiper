Display YouTube Shorts videos in an elegant swiper slider.

== Description ==

YouTube Shorts Swiper allows you to display your YouTube Shorts videos in a modern and responsive slider on your website.

**Features:**

* Easy video management - Add/delete/edit videos from admin panel
* Automatic thumbnail fetching - Automatically fetched from YouTube
* Responsive design - Separate settings for desktop, tablet, and mobile
* Click-to-play - Direct video playback on page
* Autoplay slider - Automatic sliding feature
* Shortcode support - Add to any page

**Usage:**

1. Go to "YT Shorts Swiper" menu in WordPress admin panel
2. Add video URLs
3. Customize settings
4. Add `[yt_shorts_swiper]` shortcode to any page

== Installation ==

1. Upload plugin files to `/wp-content/plugins/yt-shorts-swiper` directory
2. Activate the plugin from WordPress admin panel
3. Configure settings from "YT Shorts Swiper" menu

== Frequently Asked Questions ==

= Which YouTube URL formats are supported? =

* https://youtube.com/shorts/VIDEO_ID
* https://youtu.be/VIDEO_ID
* https://youtube.com/watch?v=VIDEO_ID

= How many videos can I add? =

There is no limit, you can add as many videos as you want.

= Where do thumbnails come from? =

Thumbnails are automatically fetched from YouTube, no API key required.

== Açıklama ==

YouTube Shorts Swiper, YouTube Shorts videolarınızı web sitenizde modern ve responsive bir slider içinde görüntülemenizi sağlar.

**Özellikler:**

* Kolay video yönetimi - Admin panelden video ekle/sil/düzenle

* Responsive tasarım - Desktop, tablet ve mobil için ayrı ayarlar
* Tıkla-oynat - Sayfada direkt video oynatma
* Autoplay slider - Otomatik kaydırma özelliği
* Shortcode desteği - Herhangi bir sayfaya ekleyin

**Kullanım:**

1. WordPress admin panelinde "YT Shorts Swiper" menüsüne gidin
2. Video URL'lerini ekleyin
3. Ayarları özelleştirin
